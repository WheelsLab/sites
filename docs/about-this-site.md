# 关于本站

建立这个网站，主要是为了在学习技术的过程中，把零散的知识整理下来，形成一套属于自己的笔记和文档。网站叫 **WheelsLab**，这个名字对我来说有几层不同的含义。

首先，**Wheels** 和我对游戏的兴趣有关。我平时很喜欢玩游戏，尤其是 3A 级别的 RPG、各种独立游戏，以及赛车类游戏，比如《极限竞速：地平线》系列。我很享受在开放世界里自由探索、看风景、发现新东西的感觉。不管是中世纪的奇幻世界、充满未来科技感的城市，还是贴近现实的角色扮演，每一次探索都会带来新的体验。因此，这里的 *Wheels* 可以理解为赛车的“轮子”，代表我作为一名玩家的身份。

其次，**Wheels** 也代表我喜欢“折腾”软硬件。从早期给 Android 手机刷机、解锁、装各种模块，到后来折腾路由器，刷 OpenWRT，再到慢慢接触 Linux 系统，研究虚拟机、容器、服务器、NAS 和博客系统。我喜欢从最基础的命令行开始，一步步理解系统是怎么工作的，甚至尝试写一些小程序、做一个简单的操作系统原型。在 Arch Linux 中，`wheel` 是一个拥有系统管理权限的用户组，这个名字也很符合我这种爱动手、爱研究底层的性格。

第三，在计算机领域里，“轮子”通常指已经做得很成熟的基础工具和系统。大家常说“不要重复造轮子”，但对学习来说，**亲手造一次轮子**，往往能真正理解背后的原理。这里的 *Wheels*，指的是操作系统、编译器、数据库、游戏引擎等这些基础软件。这个网站的目标，就是通过自己动手实现和拆解这些东西，加深对计算机技术的理解。

至于 **Labs**，意思就是“实验室”。我把学习和折腾的过程当作一次次实验，希望把其中的思考、踩过的坑和学到的东西记录下来。既是对自己知识的整理，也希望能在某种程度上对其他人有所帮助。
